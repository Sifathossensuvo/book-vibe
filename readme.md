# 📚 Book vibe

A modern and responsive book management web application built with **React**. Users can explore books, view detailed information, add books to their **Read List** or **Wishlist**, and manage their reading journey with a clean and user-friendly interface.

---

## 🚀 Live Demo

🔗 **Live Website:** *[Add your live website link here](https://book-vibe-poka.netlify.app/)*

---

## ✨ Features

* 📖 Browse and explore a collection of books
* 📚 View detailed information for each book
* ✅ Add books to the Read List
* ❤️ Add books to the Wishlist
* 🚫 Prevent duplicate book entries
* 🔔 Beautiful toast notifications using React Toastify
* 💾 Data stored in Local Storage
* 📱 Fully Responsive Design
* ⚡ Fast performance with React + Vite

---

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/your-username/boi-poka.git
```

Go to the project directory

```bash
cd boi-poka
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```
